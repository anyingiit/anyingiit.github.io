Windows 11 上安装 WSL 2

WSL的安装

0x00 前言

如无特殊说明，本文中的WSL指的是WSL 2. 本文只记录单次安装过程，对通用流程不负责。

0x01 WSL 简介

是基于Hyper-V的windows子系统，轻量（动态内存分配），非常适合执行日常跨平台开发等各种任务，尤其对低配置电脑较为友好。

0x02 前置条件

对于安装了2Ｈ22更新的windows 11系统，其默认会安装WSL 2，本文已安装该更新。

0x03 WSL安装

# If you don't have Hyper-V, excute this command
# dism.exe /online /enable-feature /featurename:HypervisorPlatform /all /norestart

# Install the wsl
wsl --install

稍等片刻后，按照提示输入安装后的Ubuntu的用户名和密码即可。 至此，就完成了WSL的安装。

0x04 结语

本文介绍了一次WSL的安装过程。
